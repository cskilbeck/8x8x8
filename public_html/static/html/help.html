Documentation

256 Pixel games are written in Javascript, so if you don't already know how to write Javascript programs, head over to CodeAdcademy and learn. Once you've done that, come back and write some games!

256 Pixel Games are played on an 16x16 grid of pixels and can display 16 colors. The coordinates start at (0,0) in the top left and go to (15,15) in the bottom right. The keys which can be used are: 'up', 'down', 'left', 'right' and 'space'. The colors available are:

 0 black
 4 darkred
 8 blue
12 pink
 1 darkgreen
 5 red
 9 lightblue
13 orange
 2 green
 6 lightred
10 purple
14 yellow
 3 lightgreen
 7 darkblue
11 magenta
15 white

setpixel(x, y, color)
Set the pixel at x,y to color. You can pass in a name (eg. 'red') or number (from 0 to 15) for the color.

This will set the top left pixel to the color blue and the bottom right pixel to red
setpixel(0, 0, 'blue');
setpixel(15, 15, 5);

rectangle(x, y, w, h, color)
Sets all the pixels in a rectangle starting at at x,y extending to x + w, y + h (inclusive) to color. As with setpixel, you can pass in a name or number for the color.

This will set 9 pixels in the top left corner to the color yellow and all the pixels on the bottom row to purple
rectangle(1, 1, 3, 3, 'yellow');
rectangle(0, 15, 16, 1, 10);

box(x, y, w, h, color)
Draws a box starting at at x,y extending to x + w, y + h (inclusive). As with setpixel, you can pass in a name or number for the color.

This will draw an orange box 6 pixels wide and 8 pixels high at (5,4) in yellow 
box(5, 4, 6, 8, 'orange');
rectangle(0, 15, 16, 1, 10);

getpixel(x, y)
Get the color of the pixel at x,y.

This will check if the color of the pixel at (4,5) is yellow
if(getpixel(4, 5) === 'yellow') {
    // ...
}

getpixeli(x, y)
Get the index of the color of the pixel at x,y.

This will check if the color of the pixel at (4,5) is green
if(getpixeli(4, 5) === 2) {
    // ...
}

clear(color)
Set all the pixels to a single color.

This will clear all the pixels to black
clear('black');

keypress()
Get the last key that was pressed.

This will check if the space bar was just pressed
if(keypress() === 'space') {
    // ...
}

keyrelease(key)
Get the last key the was released.

This will check if the left key was just released
if(keyrelease() === 'left') {
    // ...
}

keyheld(key)
Check if a key is held down.

This will check if the up key is currently being pressed
if(keyheld('up')) {
    // ...
}

reset()
Restarts the game, as if the user had clicked the 'Run' button

This will restart the game if lives is 0
if(lives === 0) {
    reset();
}

update(frame)
You provide this function. It will be called repeatedly at a rate specified by the 'speed' setting.
The parameter 'frame' is increased by 1 each time the function is called.

This is a very simple game
var x, y, c;

x = 0;
y = 0;
c = 1;
clear(0);   // code outside functions is run once when game starts

function update(frame) {
    if(keyheld('right')) {
        x = x + 1;
        if(x === 8) {
            clear('black');
            x = 0;
            y = y + 1;
            if(y === 8) {
                y = 0;
            }
        }
    }
    if(keypress() === 'space') {
        c = c + 1;
        if(c === 8) {
            c = 1;
        }
    }
    setpixel(x, y, c);
}
