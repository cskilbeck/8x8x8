Documentation
8x8x8 Games are played on an 8x8 grid of pixels and can display 8 colors. The coordinates start at (0,0) in the top left and go to (7,7) in the bottom right. The keys which can be used are: 'up', 'down', 'left', 'right' and 'space'. The colors available are:

0 - Black
1 - Red
2 - Green
3 - Yellow
4 - Blue
5 - Magenta
6 - Cyan
7 - White

Functions:
set(x, y, color)
Set the pixel at x,y to color.

This will set the top left pixel to the color cyan
set(0, 0, 6);

get(x, y)
Get the color of the pixel at x,y.

This will check if the color of the pixel at (4,5) is yellow
if(get(4, 5) === 3) {
    // ...
}

clear(color)
Set all the pixels to a single color.

This will clear all the pixels to black
clear(0);

pressed(key)
Check if key was just pressed.

This will check if the space bar was pressed since the last time update() was called
if(pressed('space')) {
    // ...
}

released(key)
Check if key was just released.

This will check if the left key was released since the last time update() was called
if(released('left')) {
    // ...
}

held(key)
Check if key is held down.

This will check if the up key is currently being pressed
if(held('up')) {
    // ...
}

update(frame)
You provide this function, it will be called repeatedly about 60 times every second.
The parameter 'frame' is increased by 1 each time the function is called.

This is a very simple game
var x, y, c;

x = 0;
y = 0;
c = 1;
clear(0);   // code outside functions is run once when game starts

function update(frame) {
    if(held('right')) {
        x = x + 1;
        if(x === 8) {
            clear(0);
            x = 0;
            y = y + 1;
            if(y === 8) {
                y = 0;
            }
        }
    }
    if(pressed('space')) {
        c = c + 1;
        if(c === 8) {
            c = 1;
        }
    }
    set(x, y, c);
}
