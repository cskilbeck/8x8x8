8x8x8 - Documentation
 
 
set(x, y, color)
Set the pixel at x,y to color. Coordinates are (0,0) = top left, (7,7) = bottom right. Colors are: 0 = black, 1 = red, 2 = green, 3 = yellow, 4 = blue, 5 = magenta, 6 = cyan, 7 = white
 
clear(color)
Set all the pixels to color.
 
pressed(key)
Check if key was just pressed. Keys are 'left', 'right', 'up', 'down', 'space'.
 
released(key)
Check if key was just released. Keys are 'left', 'right', 'up', 'down', 'space'.
 
held(key)
Check if key is currently held down. Keys are 'left', 'right', 'up', 'down', 'space'.
 
update(frame)
You provide this function, it will be called every frame in a loop. Parameter 'frame' increases by 1 each time the function is called.
